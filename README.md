# Aplicação de teste pra FullStack developer

## Baixe o código

```
$ git clone https://github.com/marccalixto/PortalTeleMedicina.git

 ## WebApi project
$ cd PortalTeleMedicina
$ cd WebApi
$ open solution file in Visual Studio
$ install dependencies
//run the app

 ## WebClient project
$ cd Client
$ npm install
//run the app
$ ng start
```
